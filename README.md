# 🐳 Docker Task

A beginner-friendly Docker project focused on understanding containerization, Docker images, and running applications inside isolated environments.

This project demonstrates the basics of:
- Docker
- Dockerfile creation
- Building Docker images
- Running containers
- Port mapping
- Container management

---

# 📌 Project Objective

The goal of this project is to learn how Docker works by creating and managing containers in a simple and practical way.

This repository helps in understanding:
- How applications run inside containers
- How Docker simplifies deployment
- How to create reproducible environments
- Basic DevOps workflow using Docker

- Linux Commands
- Git & GitHub
